# Remindy

แอพตัวอย่าง Remindy — แอพเตือนความจำแบบง่าย (Flutter, Provider, Material 3)

สรุปสั้น ๆ
- โครงสร้างหลัก: `lib/` มี `main.dart`, `app.dart`, `models/`, `providers/`, `screens/`, `widgets/`
- State management: `provider`
- แพ็กเกจสำคัญ: `provider`, `intl`, `uuid`, `flutter_slidable`, `fl_chart`, `google_fonts`

Prerequisites (Windows)
- Flutter SDK ติดตั้งและอยู่ใน PATH
- Android Studio + Android SDK (Platform, Platform-tools, Command-line tools)
- AVD (emulator) สร้างไว้ใน Android Studio (เช่น Pixel_7)

คำสั่งใช้งาน (จากโฟลเดอร์โปรเจค `D:\8\remindy`)

1) ติดตั้ง dependencies:
```powershell
flutter pub get
```

2) รันบน Windows desktop:
```powershell
flutter run -d windows
```

3) รันบน Chrome (เว็บ):
```powershell
flutter run -d chrome
```

4) รันบน Android emulator (ตัวอย่าง):
```powershell
flutter emulators
flutter emulators --launch <emulatorId>
flutter run -d <deviceId>
```

ปัญหา build บน Android (ถ้าพบ)
- หากเจอข้อความคล้าย: "NDK ... did not have a source.properties file" หรือ Gradle build ล้มเหลว เป็นไปได้ว่าไฟล์ NDK ถูกดาวน์โหลดไม่สมบูรณ์
- วิธีแก้ (แนะนำ):
  1. ปิด emulator/Android Studio
  2. ลบโฟลเดอร์ NDK ที่ระบุใน error เช่น `C:\Users\<you>\AppData\Local\Android\Sdk\ndk\28.2.13676358`
  3. เปิด Android Studio → SDK Manager → SDK Tools → ติ๊ก `NDK (Side by side)` แล้วกด Apply เพื่อให้ดาวน์โหลดใหม่
  4. หรือรัน `flutter run` อีกครั้ง แล้ว Gradle/Android plugin จะดาวน์โหลด NDK ให้ (ต้องเชื่อมต่ออินเทอร์เน็ต)
